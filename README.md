# Generator Agent for GitHub Copilot

## Installation

1. Clone the repository:
   git clone https://github.com/ishwaryaramesh200-byte/Generator-Agent.git

2. Copy the `.github/agents` folder into your project root.

## Usage

Open GitHub Copilot Chat.
Select the "Generators" agent.
Start prompting.

## Features

- Static generator
- Dynamic generator
- Conditional generator
- Reference generator
- Remote generator
